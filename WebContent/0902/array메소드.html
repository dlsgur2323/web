요소 찾기

arr = ["사과","바나나","참외","복숭아","사과","자두","수박","바나나"];
indexOf(item, start) : item의 위치를 찾는다.
lastIndexOf(item, start) : 마지막 item의 위치를 찾는다.

배열에 추가/삭제

arr = ["사과","바나나","참외","복숭아","사과","자두","수박","바나나"];
push(item) : 배열의 맨 마지막으로 추가
pop() : 배열의 마지막 요소를 제거

배열에 추출/수정

arr = ["사과","바나나","참외","복숭아","사과","자두","수박","바나나"];
slice(start, end) : start~ end-1 까지의 요소를 추출
splice(idx, n, a, b, c) : idx 위치에서 n개를 삭제하고 a, b, c 를 추가

배열 합치기 / 문자열 변환

arr = ["사과","바나나","참외","복숭아","사과","자두","수박","바나나"];
arr.concat(a) : arr 배열과 a 배열이 하나의 배열로 합쳐진다.
arr.join() : arr 배열을 문자열로 변환한다.
